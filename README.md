# Metre BTP Niger

Site statique HTML, CSS et JavaScript pur pour calculer les quantites de materiaux BTP : beton, ciment, sable, gravier, briques, enduit, peinture, carrelage, dalle, fondation, poteaux, poutres, ferraillage, toiture, cloture et estimation de maison.

## Publication GitHub Pages

1. Envoyer tous les fichiers a la racine d'un depot GitHub.
2. Ouvrir `Settings > Pages`.
3. Choisir la branche principale et le dossier `/root`.
4. Publier.

Le site fonctionne sans backend. Les prix personnalises sont sauvegardes dans `localStorage`.
